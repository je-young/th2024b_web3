"JPA 과제4 : 비품 관리 CRUD
[필수패턴] : BaseTime , toEntity/toDto , Spring MV2 3tire : Controller/Serivce/Respository
[기능구현]
    1.비품 등록 기능
        · 사용자는 새로운 비품 정보를 등록할 수 있어야 한다.
        · 등록 시 필요한 정보: 비품명(name), 비품 설명(description), 수량(quantity)
        · 등록 시간(createdDate)과 수정 시간(modifiedDate)은 자동으로 기록되어야 한다.
    2.전체 비품 조회 기능
        · 저장된 모든 비품 목록을 조회할 수 있어야 한다.
    3.개별 비품 조회 기능
        · 특정 비품의 상세 정보를 id를 통해 조회할 수 있어야 한다.
    4.비품 수정 기능
        · 수정 시 특정 비품의 id를 기준으로 수정하며, 수정 시간은 자동으로 갱신되어야 한다.
    5.비품 삭제 기능
        · 특정 비품의 id를 기준으로 해당 비품 정보를 삭제할 수 있어야 한다.
    6.페이징 기능이 적용된 비품 목록 조회
        · 비품 목록을 페이지 단위로 조회할 수 있어야 하며,
        페이지 번호(page)와 페이지당 항목 수(size)를 입력받아 처리한다.
